### 友元函数
能够访问类的私有和保护成员
### 特点：
- 不是类的成员函数
- 需要在类内部用==<sub></sub>​friend​<sub></sub>==关键字声明
- 定义在类外部，不需要使用类名和作用域解析运算符
声明方式
```c++

class MyClass {

private:

int secret;

public:

// 声明友元函数

friend void showSecret(MyClass obj);

};

// 定义友元函数（不需要MyClass::）

void showSecret(MyClass obj) {

cout << "Secret is: " << obj.secret; // 可以访问私有成员

}
```
**注意：**
1.  友元是单向的（A声明B，不能意味B把A当友元）
2.  不能继承
3.  不能传递
4.  过度使用会破坏封装性
